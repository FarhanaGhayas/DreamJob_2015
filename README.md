# DreamJob
# University ASP.NET Web Forms Project

Old university project upgraded for learning and improvement. It is a role-based Job Portal web application built with **ASP.NET WEB FORMS** that connects **Job Seekers** and **Recruiters** on a single platform.  
Recruiters can register companies, post jobs, and manage applications, while Job Seekers can search jobs, upload CVs, and manage their inbox.
